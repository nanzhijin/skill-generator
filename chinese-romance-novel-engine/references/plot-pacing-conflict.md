# 情节节奏与冲突设计

## Background
评估故事事件的编排、节奏控制以及核心冲突的设计，确保剧情推进有吸引力且符合中国言情读者对情节密度的期待（如强情节或细腻日常的平衡）。参考《心跳引擎》中引擎驱动情节加速的叙事技巧。

**《心跳引擎》参考模式（如可获取）**：该小说常以情感爆发点作为情节加速的引擎，冲突升级与情感揭露同步，转折往往由人物内在欲望驱动而非外部巧合。若可获取，请对比本文段的节奏设计；否则使用“事件-情感咬合”这一高标准评判，并注明未提供原文。

## Analysis Dimensions
1. 节奏松紧度：叙事速度是否与情感强度匹配，有无拖沓或仓促的段落，关键场景是否得到充分渲染。
2. 冲突层次：是否存在至少两个层次的冲突（如内在冲突vs外部阻碍），冲突是否与感情主线紧密咬合。
3. 钩子与悬念：章节/段落内是否设置了足够的悬念或继续阅读的动机（钩子）。
4. 转折自然度：情节转折是否建立在已有细节之上，避免“机械降神”式的生硬转折。
5. 心跳引擎对齐度（若可用）：检查转折是否由情感动机引爆，节奏是否与情感曲线同步；若无则跳过。

## Evaluation Criteria
基于网络小说编辑常用标准，结合完读率导向：
- **高吸引力 (85-100)**：节奏完美服务情感，冲突强烈且自然，钩子密集有效，能令读者不愿中断。
- **平稳 (60-84)**：节奏缓急有序但缺乏爆发点，冲突存在但力度不足，钩子偶有但不够连续。
- **乏味 (0-59)**：节奏混乱，冲突单薄或牵强，缺乏推动阅读的动力。
按维度扣分，最终折合分数。

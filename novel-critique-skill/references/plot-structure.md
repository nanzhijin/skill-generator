# 情节结构分析

## 背景
分析小说的总体叙事弧线与结构，评估情节的完整性、逻辑性与因果关系，发现漏洞与牵强之处。仅针对已提供的文本段落，不引入外部假设。

## 分析维度
1. 叙事弧线完整性：是否存在清晰的开端、发展、高潮、结局（或符合该体裁的结构范式），各部分比例是否恰当。
2. 因果关系逻辑：事件推进是否基于前因后果，角色决策与转折是否合理可信。
3. 伏笔与收束：关键伏笔是否在后文得到妥善揭示与回收，是否存在遗忘或偏离。
4. 次级情节整合：支线是否与主线有机交织，是否推动主题或人物成长，有无冗余或脱节。

}

## 评估标准
- 优秀 (90-100): 情节完整、逻辑严密，伏笔全部得当回收，次级情节无任何冗余，且与主线形成共鸣。
- 良好 (75-89): 结构基本完整，仅有1-2处轻微瑕疵；伏笔大多回收，次级情节整体协调。
- 一般 (50-74): 存在明显漏洞或逻辑牵强，伏笔回收率低（小于50%）或次级情节明显干扰主线。
- 差 (0-49): 情节混乱不堪，因果链断裂严重，大量未回收伏笔，次级情节与主线毫无关联或破坏整体。
注：不同评审者独立打分，对相同文本在以上四个等级上的判断一致率超过80%。
